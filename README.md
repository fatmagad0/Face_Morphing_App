# 🎭 AI Face Morpher Web App
A professional web application that uses **Dlib** and **OpenCV** to create a seamless morphing animation between two faces.

## ✨ Features
- **Glassmorphism UI:** Modern and creative web interface.
- **Delaunay Triangulation:** High-quality face warping.
- **Animated Results:** Generates smooth GIF transitions.

## 🛠️ How to run
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`.
3. Download `shape_predictor_68_face_landmarks.dat` and place it in the root folder.
4. Run `python main.py`.
